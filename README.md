# CRE Underwriting Model

A commercial real estate underwriting model built with React, Vite, and Tailwind CSS.

## Features

- **Acquisition Inputs**: Purchase price, cap rate, closing costs.
- **Growth Strategy**: Annual vs. Step-up growth modeling.
- **Debt & Exit**: LTV, interest rate, amortization, and exit scenarios.
- **Real-time Calculations**: IRR, Equity Multiple, Cash-on-Cash return.
- **Pro Forma**: detailed 10-year cash flow projections.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Deployment

This project is configured for deployment on Vercel.

1.  Push to GitHub.
2.  Import project in Vercel.
3.  Deploy (settings are pre-configured in `vercel.json`).
